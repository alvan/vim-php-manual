===============================================================================
*Description*

  wddx_packet_start ([ string $comment ] ) : resource~

Start a new WDDX packet for incremental addition of variables. It automatically
creates a structure definition inside the packet to contain the variables.

===============================================================================
*Parameters*

{comment}

  An optional comment string.

===============================================================================
*Return* *Values*

Returns a packet ID for use in later functions, or FALSE on error.
