===============================================================================
*Description*

  unixtojd ([ int $timestamp = time() ] ) : int~

Return the Julian Day for a Unix {timestamp} (seconds since 1.1.1970), or for
the current day if no {timestamp} is given. Either way, the time is regarded as
local time (not UTC).

===============================================================================
*Parameters*

{timestamp}

  A unix timestamp to convert.

===============================================================================
*Return* *Values*

A julian day number as integer.
