phpdbg_color

(PHP 5 >= 5.6.0, PHP 7)

phpdbg_color â€” Sets the color of certain elements

===============================================================================
*Description*

  phpdbg_color ( int $element , string $color ) : void~

Set the {color} of the given {element}.

===============================================================================
*Parameters*

{element}

  One of the PHPDBG_COLOR_* constants.

{color}

  The name of the color. One of white, red, green, yellow, blue, purple, cyan
  or black, optionally with either a trailing -bold or -underline, for
  instance, white-bold or green-underline.

===============================================================================
*Return* *Values*

No value is returned.
