fann_set_cascade_activation_functions -- Sets the array of cascade candidate
activation functions

  bool fann_set_cascade_activation_functions(resource ann, array cascade_activation_functions)~

Sets the array of cascade candidate activation functions.

See |fann_get_cascade_num_candidates| for a description of which candidate
neurons will be generated by this array.

Neural network resource.

The array of cascade candidate activation functions.

Returns TRUE on success, or FALSE otherwise.
