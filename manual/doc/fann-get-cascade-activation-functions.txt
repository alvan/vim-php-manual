fann_get_cascade_activation_functions -- Returns the cascade activation
functions

  array fann_get_cascade_activation_functions(resource ann)~

The cascade activation functions array is an array of the different activation
functions used by the candidates

See |fann_get_cascade_num_candidates| for a description of which candidate
neurons will be generated by this array.

The default activation functions are FANN_SIGMOID, FANN_SIGMOID_SYMMETRIC,
FANN_GAUSSIAN, FANN_GAUSSIAN_SYMMETRIC, FANN_ELLIOT, FANN_ELLIOT_SYMMETRIC.

Neural network resource.

The cascade activation functions, or FALSE on error.

|fann_get_cascade_activation_functions_count|
