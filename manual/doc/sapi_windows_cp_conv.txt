==============================================================================
*Description*

    string sapi_windows_cp_conv ( int|string $in_codepage , int|string~
    $out_codepage , string $subject )~

Convert string from one codepage to another.

==============================================================================
*Parameters*

{in_codepage}

    The codepage of the {subject} string. Either the codepage name or
    identifier.

{out_codepage}

    The codepage to convert the {subject} string to. Either the codepage name
    or identifier.

{subject}

    The string to convert.

==============================================================================
*Return* *Values*

The {subject} string converted to {out_codepage}, or NULL on failure.

==============================================================================
*Errors/Exceptions*

This function issues E_WARNING level errors, if invalid codepages are given,
or if the subject is not valid for {in_codepage}.
